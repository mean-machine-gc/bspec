# System UbiSpec v1-0 â€” Schema Reference

## SystemSpec Object

System UbiSpec: top-level map of modules, contexts, and cross-module flows.

| Field | Type | Description |
|-------|------|-------------|
| `ubispec` | `"system/v1.0"` | **Required.** Format identifier and version. |
| `system` | [PascalName](#pascalname) | **Required.** System name. PascalCase. |
| `description` | `string` (optional) | _Optional._ Brief description of the system. |
| `modules` | List of [Module](#module) | **Required.** Modules in this system. Each groups related deciders under a bounded context. |
| `flows` | [Flow](#flow)[] (optional) | _Optional._ Cross-module flows. Each connects an event from one module to a command on another. |

## Flow Object

Cross-module flow: event from one module triggers a command on another.

| Field | Type | Description |
|-------|------|-------------|
| `event` | [PascalName](#pascalname) | **Required.** Event name that crosses the module boundary. PascalCase. |
| `from` | [PascalName](#pascalname) | **Required.** Source module that publishes the event. |
| `triggers` | [PascalName](#pascalname) | **Required.** Command name invoked on the target module. PascalCase. |
| `on` | [PascalName](#pascalname) | **Required.** Target module that receives the command. |
| `trigger` | [FlowTrigger](#flowtrigger) (optional) | _Optional._ `automated` (default) or `policy`. Policy flows require `actor`. |
| `actor` | `string` (optional) | _Optional._ Role or persona expected to fulfill a policy flow. Required when trigger is `policy`. |

## FlowTrigger

`automated`: infrastructure handles delivery. `policy`: human or external process must act.

| Value | Description |
|-------|-------------|
| `"automated"` | Allowed value |
| `"policy"` | Allowed value |

## Module Object

Module: a group of closely related deciders sharing a consistency boundary.

| Field | Type | Description |
|-------|------|-------------|
| `name` | [PascalName](#pascalname) | **Required.** Module name. PascalCase. Referenced by process specs and flows. |
| `context` | [PascalName](#pascalname) | **Required.** Bounded context this module belongs to. PascalCase. Grouping label for related modules. |
| `deciders` | List of [PascalName](#pascalname) | **Required.** Deciders in this module. Each must have a corresponding lifecycle spec. |
| `description` | `string` (optional) | _Optional._ Brief description of the module's responsibility. |

## PascalName

PascalCase identifier.

| Constraint | Value |
|------------|-------|
| **Type** | `string` |
| **Pattern** | PascalCase identifier |
